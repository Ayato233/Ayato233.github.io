二、简答题

  1. byte、short、int、long、double、float、bool、char

    1. byte b = 1;
    2. short s=1;
    3. int i=1;
    4. long l = 1L;
    5. double d=3.14d;
    6. float f=3.14f;
    7. bool flag=true;
    8. char cha =’a’;

  2. 单支if、if-else、多重if、嵌套if、switch选择结构

    1. ```
      if(布尔表达式){//代码块}
    2. ```
      if(布尔表达式){//代码块}else if(布尔表达式){//代码块}else{//代码块}
    3. ```
      switch(表达式){
      case 常量表达式1:
      //代码
      break;
      case 常量表达式2:
      //代码
      break;
      default:
      //代码
      break;
      }
    4. ```
      do{//循环体}while(布尔表达式);
    5. foreach(类型 变量 in 数组){//循环体}

  3. 代码可以被重复利用，具有很好的安全性，不需要关心具体实现方式，直接拿来使用就好

    封装就是将一段代码提取成一个公开方法或者私有方法或者一个类里面，它可以完成某个具体的功能操作，DBHelper就是个很好的例子，设置私有化字段提供公开接口也可称为封装

  4. 值传递就是调用方法时把实际参数复制一份传到方法里面，这样在方法里对值做的修改不会影响原来的值（原来的值：传递之前的实际参数）;

  5. DataSet数据集和.NET Framework 数据提供程序

    核心对象为四个：connection对象负责连接数据库、command对象负责对数据库执行命令、DataReader对象负责从数据源读取数据、DataAdapter对象负责数据库和数据集的联系。

  6. Read() 方法,用来读取数据源，判断是否为空，前进到下一条数据

    close() 方法，用来关闭datareader的连接。

  7. ExcuteScalar()、查询单个值方法

    ExcuteNonQuery()、执行增删改

    ExecuteReader()、查询多个值

  8. 定义类，，添加一个类，写好类名，注意命名规范，类是引用数据类型，类是实例的模板。

    定义方法，，在一个类里可以定义有参方法和无参方法，可以定义多个方法，顺序无所谓，方法不调用就不会执行
