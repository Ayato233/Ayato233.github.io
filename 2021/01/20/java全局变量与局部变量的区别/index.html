全局变量(也称成员变量)与局部变量的特点

1. 作用域不同

局部变量的作用域仅限于定义它的方法
成员变量的作用域在整个类内部都是可见的

2. 初始值不同

成员变量在声明未赋值时有初始值(即默认值)
局部变量在声明未赋值时无初始值

3. 变量名使用特点

在同一个方法中，不允许有同名局部变量
在不同的方法中，可以有同名局部变量

4. 就近原则

在同一个类中，成员变量和局部变量同名时，局部变量具有更高的优先级

此时如果想使用成员变量则需要在变量前加this关键字
